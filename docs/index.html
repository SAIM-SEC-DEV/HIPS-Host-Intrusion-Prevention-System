Verified Installation
Follow these steps in order
1

Database Setup

Create the database and import the schema.

CREATE DATABASE hips_db;
USE hips_db;
SOURCE database/schema.sql;
ℹ️ Optional: The fourth migration step (database/migration.sql) was not required during testing and may be skipped unless additional updates are provided.
2

Server Setup

Copy the entire server folder into XAMPP's htdocs directory.

xampp/
└── htdocs/
    └── server/          ← folder name must remain "server"
        ├── dashboard/
        ├── api/
        ├── config/
        └── ...
ℹ️ Apache serves only files inside htdocs. The folder name server is required because the agent configuration references it.
3

Agent Configuration

Rename the example config and set the required values.

// Rename this file:
config/hips-agent.properties.example
// To:
config/hips-agent.properties
server.url         = http://localhost/server
auth.token         = # automatically generated — do not modify
agent.owner        = Office-PC
watch.directories  = C:\Users\<username>\Desktop
ℹ️ auth.token is automatically generated. Do not change it. agent.owner can be any descriptive name.
4

Recompile Agent

Run the recompilation script to apply the new configuration.

recompile-agent.bat
💡 Recompilation updates the agent binary with your settings before execution.
5

Run the Server

Start all server components with a single script.

Start HIPS Server.bat
ℹ️ This starts Apache, MySQL, the dashboard, and the server API. After startup, access the dashboard at http://localhost/server/dashboard.
